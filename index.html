📖 这是什么？
梦角传呼机是一个沉浸式AI角色扮演聊天工具。你可以创建独特的角色，通过复古传呼机风格的界面与他们交流，仿佛连接着电子世界中真实存在的生命。
🚀 快速上手
  • 点击右上角 ⚙ 进入设置，在 API 页面配置你的API密钥
  • 在 人设 页面设置角色名称和性格描述
  • 输入消息后点击 发送，再点 接收 获取AI回复
  • 点击左上角角色名可 切换/创建 多个角色
  • 点击状态栏小图标可展开查看 心情/状态/余额
✨ 功能亮点
  • 多角色系统 — 创建多个角色，独立对话存档
  • 空间感知 — 角色能感知所处位置和环境
  • 情绪系统 — 角色有心情波动，影响回复风格
  • 经济系统 — 角色有自己的收支和电子币余额
  • 记忆系统 — 自动总结对话，核心记忆手动编辑
  • 礼物/转账 — 送礼物、发起付款请求
  • 一起听歌/看书 — 共享活动，角色能感受到
  • 纪念日 — 记录重要日期，临近时角色会提起
  • 主题皮肤 — 8套预设配色方案可选
  • 可视化地图 — 空间系统星空地图，点击移动
💡 小提示
  • 长按消息可 编辑/删除/收藏/重说
  • 点击 + 号可发图片、送礼物、听歌、看书、发表情包
  • 定期在 数据 页面导出备份，防止数据丢失
  • 世界书中的内置提示词可以开关，自定义AI行为
  • 重要：每次对话后请点击顶部闪烁的 🔄 按钮重连，确保下次对话正常
